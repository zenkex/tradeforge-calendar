### 项目名称

**P&L Calendar** - 本地交易盈亏日历记录工具

### 项目目标

开发一个美观、专业、纯本地运行的交易盈亏日历网页，专门用于外汇/期货交易者记录每日 P&L。完全仿照 Lucid Trading 的 Calendar 风格设计，强调清晰、直观、交易氛围。

### 核心需求

- **纯前端实现**：单个 HTML 文件内嵌所有 CSS 和 JS
- **数据 100% 本地存储**：使用 `localStorage`（key: `pnlCalendar`），不发送任何数据到服务器
- **月度视图**：默认显示当前月，支持上/下月切换，一键回到今天
- **周日作为每周起始**：日历第一列为周日，最后一列为周六
- **周末自动识别**：周六、周日显示为休市状态（不可输入，特殊样式；周六虽然不可编辑但可显示周合计）
- **日期颜色规则**：
  - **绿色**：当日盈利（P&L > 0）
  - **红色**：当日亏损（P&L < 0）
  - **灰色**：未交易或 P&L = 0
- **手动输入**：点击工作日日期弹出 Modal 输入当日盈亏金额（支持正负数）和备注

### UI/UX 要求（仿照 Lucid Trading 风格）

1. **整体风格**：
   - 深色交易主题（zinc-950 / zinc-900 背景）
   - 专业金融感，英文使用 Space Grotesk 字体，中文使用 Noto Sans SC 字体
   - P&L 金额数字使用 JetBrains Mono 等宽字体
   - 卡片式布局，适当圆角和阴影
   - 字体从 Google Fonts 加载，设置 `font-display: swap` 确保加载期间文字可见

2. **日历布局**：
   - 顶部：年月切换按钮（`←` `→`） + "今天"快捷按钮
   - 表头：日、一、二、三、四、五、六（周日和周六表头文字淡化）
   - 日期格子：大号日期数字（固定灰色 `text-zinc-400`，不随盈亏变色） + 当日 P&L 金额
   - 格子背景半透明着色：盈利 `bg-emerald-500/10`，亏损 `bg-red-500/10`，无交易 `bg-zinc-800/30`
   - 周一至周五格子边框为 2px（`border-2`），周末格子无边框
   - 今日日期格子右上角显示一个琥珀色小圆点（`bg-amber-400`），便于快速定位
   - 今日日期格子有发光圆环效果（`box-shadow` ring）

3. **交互**：
   - 点击工作日日期打开 Modal 输入 P&L
   - Modal 中显示日期、金额输入框（前缀 `$` 符号，type="number" step="0.01"）、备注输入框、保存/取消/删除按钮
   - 点击周末日期无反应
   - 键盘快捷键：`Esc` 关闭 Modal，`Enter` 在 Modal 中保存，`←` `→` 在日历视图中切换月份

4. **周合计**：
   - 周六格内显示当周（周日 ~ 周六）所有日期 P&L 的累计总和，前缀 `Σ` 符号
   - 周合计跨月计算：若周的周日在上月，从上月数据中读取
   - 仅当周六在当前显示月份内时才显示周合计
   - 周合计金额颜色根据总和正负显示绿/红，字号与当日 P&L 金额相同

5. **金额显示格式**：
   - 所有 P&L 金额保留两位小数（如 `$1,250.75`、`-$450.00`）
   - 正数不显示 `+` 号，只显示 `$` 前缀（如 `$500.00`）
   - 负数显示 `-` 号和 `$` 前缀（如 `-$500.00`）
   - 日历格子内金额因空间有限，大数值使用 k/M 缩写（如 `$1.25k`、`-$2.50k`）
   - 统计面板使用完整格式（带千位分隔符）

### 技术实现规格

#### 数据结构（localStorage key: `pnlCalendar`）

```json
{
  "2026-05-01": {
    "pnl": 1250.75,
    "note": "日内突破交易"
  },
  "2026-05-02": {
    "pnl": -450.00,
    "note": "止损调整不当"
  }
}
```

#### CSS 颜色规范

- 盈利绿：`text-emerald-400` / `bg-emerald-500/10` / `border-emerald-500/20`
- 亏损红：`text-red-400` / `bg-red-500/10` / `border-red-500/20`
- 休市/无交易：`bg-zinc-800/30` / `text-zinc-400`
- 周末：`bg-zinc-900/30` / `text-zinc-600`（无边框，cursor: default）
- 今日标记：`bg-amber-400`
- 统计占位符/空数据：`text-zinc-500`

#### 核心函数

- `fmtDate(y, m, d)` — 格式化为 `YYYY-MM-DD` 字符串
- `fmtPnL(value, compact)` — 格式化 P&L 金额，compact 模式用于格子内显示
- `getDaysInMonth(y, m)` — 获取指定月份天数
- `getFirstDayOffset(y, m)` — 计算当月 1 日是周几（0=周日）
- `isWeekend(y, m, d)` — 判断是否周六或周日
- `getWeekTotal(y, m, d)` — 以周六日期为锚点，计算当周（周日至周六）P&L 总和（跨月读取）
- `loadData()` / `saveData()` — localStorage 读写
- `getEntry(dateStr)` / `setEntry(dateStr, pnl, note)` / `deleteEntry(dateStr)` — 数据 CRUD
- `calcTradeStats(entries)` — 计算胜率、平均盈亏、盈亏比、最佳/最差日、连续盈亏等
- `renderCalendar()` — 渲染日历网格
- `renderStatsCards()` — 渲染本月/今年/全部总盈亏卡片
- `renderAdvancedStats()` — 渲染高级统计面板
- `drawChart()` — Canvas 绘制本月每日盈亏柱状图
- `openModal(dateStr)` / `closeModal()` / `saveModalEntry()` / `deleteModalEntry()` — Modal 交互
- `exportData()` / `importData()` — JSON 导入导出
